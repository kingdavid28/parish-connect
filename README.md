
  # Build Parish Connect App

  This is a code bundle for Build Parish Connect App. The original project is available at https://www.figma.com/design/8K7W2MLfeJmzhUd5Eg0PnJ/Build-Parish-Connect-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  

  Auto-awarded points (hooked into existing routes):

Action	Points
Create a post	+10
Add a comment	+5
Receive a like	+2
Receive a follow	+3
Receive kudos	+15
Daily login	+5
